Application Process

Applicant himself must access the https://msc.priisms.online, as New Applicant
  1. Create New account using active and correct email address, only one account is allowed;
  2. Browse Image to attach ID picture;
  3. Choose File to upload Report Card;
  4. Applicant select the preferred the preferred Entrance Test Date from the options provided;
  5. Before clicking the ‘I Agree’ button, Applicant must review carefully the correctness of entries on information and must have read the Data Privacy Policy;
  6. Pooling Number will be generated for a successfully submitted application;
  7. An email regarding status of application will be in 3 days;
  8. Download and print the Entrance Test Permit sent via email, and bring on the day of testing.
